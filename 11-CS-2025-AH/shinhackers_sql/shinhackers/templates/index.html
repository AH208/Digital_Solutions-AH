Part 1

  • Games played in april

    Select all from the Games table where the month is april.

    SELECT * FROM Games WHERE Month = 4
  • Games that we won

    Select the team from the game table where our score is higher than their score.

    SELECT Team FROM Games WHERE Ours > Theirs
  • All Games Played

    Select all from the game table.

    SELECT * FROM Games
  • Details of games played in May

    Select the day, month, team, from the game table where the month is May.

    SELECT Day, Month, Team FROM Games WHERE Month = 5
  • Scores of games versus Bellyfloppers and Kneeknockers.

    Select the team, our score, their score from the game table where the team we were versus is Bellyfloppers or Kneeknockers.

    SELECT Team, Ours, Theirs FROM Games WHERE Team IN ('Bellyfloppers', 'Kneeknockers')
  • Games that we won.

    Select the frequency number of when our score is more than their score from the game table. 

    SELECT COUNT(*) FROM Games WHERE Ours > Theirs
  • Our biggest winning margin game.

    Select the max number collected when our score is subtracted from their score in the game table.

    SELECT max(Ours-Theirs) FROM Games
  • All games ordered by date

    Select all from the game table sorted by month then day.

    SELECT * FROM Games ORDER BY Month, Day
  • All games ordered by date (in a table)

    (In a table) Select all from the game table sorted by month then day.

    SELECT * FROM Games ORDER BY Month, Day

  • Total points scored in all games 
    SELECT COUNT(*), SUM(Ours), SUM(Theirs) FROM Games
  • Teams that we have beaten by 10 
    SELECT Team FROM Games WHERE Theirs - Ours >= 10
  • All games ordered by score 
    SELECT * FROM Games ORDER BY Ours DESC
  • All games ordered by score (in a table) 
    SELECT * FROM Games ORDER BY Ours DESC
  • All games played in april (in a table)
